# CCXT-SNIPPET

## MARKET Close position Futures with ccxt 
```python
close_sell_position = exchange.create_order(
        symbol=SYMBOL,
        type="MARKET",
        side="buy",       # buy//sell
        amount=VOLUME,
        params={"reduceOnly": "true"},
    )
```
